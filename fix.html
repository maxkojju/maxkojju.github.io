Solutions to common problems

If the website isn't working properly, here are the most common problems and their solutions:

Images don't load

If images don't load, you'll see error text or this placeholder:

Images from E926 do not load

The E926 API is very unstable.

Solution

Try using VPN or select E621 and add "rating:s" to the tags.

Images from E621 are not loaded

Your internet connection is probably too slow or E621 is blocked in your region.

Solution

Try turning on the VPN or changing the network.

The mobile app opens the links crookedly

The solution is not to use the mobile app. It's shit and it's suitable in a narrow range of situations.

The site is not loading

I don't know how you got to this page.

Solution

Try to enable VPN or open it using another link: maxkojju.github.io
